# CS 503: Object-Oriented Programming

## Overview

This repository contains my coursework, programming assignments, and final project completed for CS 503: Object-Oriented Programming at Southern New Hampshire University.

The purpose of this course is to develop a strong foundation in object-oriented programming through the design and implementation of Java applications. Throughout the course, I apply software engineering principles, algorithmic problem solving, and object-oriented design techniques to create maintainable and efficient software solutions.

## Learning Objectives

Throughout this course, I focus on:

- Applying object-oriented programming principles in Java
- Designing software using UML class diagrams
- Implementing encapsulation, inheritance, composition, and polymorphism
- Organizing applications using MVC and DAO architectures
- Applying common software design patterns
- Building reusable interfaces and abstract designs
- Implementing exception handling for robust applications
- Developing CRUD applications using object-oriented principles
- Following professional software development and version control practices with Git and GitHub
